## 13. 交互规范总结

### 设计原则
1. **角色适配** - 不同角色看到不同的默认视图和操作入口
2. **最少点击** - 核心操作（状态变更、评论）在当前页面完成，不跳转
3. **信息继承** - 任务详情自动展示前序阶段产出，减少翻找
4. **移动优先** - 现场人员的高频操作（拍照、签到、状态更新）针对手机优化

### 组件库建议
- 基础组件: Element Plus (Vue 3)
- 甘特图: dhtmlxGantt 或 frappe-gantt
- 看板拖拽: vue-draggable / dnd-kit
- 图表: ECharts
- 富文本/评论: Tiptap
- 电子签名: signature_pad
